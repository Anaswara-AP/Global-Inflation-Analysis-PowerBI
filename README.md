# 🌍 Global Inflation Analysis Using Power BI

## 📌 Project Overview

Inflation is a significant economic indicator that affects businesses, consumers, and policymakers worldwide. This project analyzes global inflation data from 1980 to 2024 and transforms raw data into an interactive Power BI report to identify trends, regional patterns, and high-inflation economies.

---

## 🎯 Objectives

- Analyze global inflation trends over time.
- Compare inflation rates across countries and regions.
- Identify countries with high inflation levels.
- Categorize inflation rates as High, Medium, and Low.
- Create an interactive dashboard for better decision-making.

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
- CSV Dataset

---

## 🔄 Data Preparation

The following transformations were performed using Power Query:

- Imported raw inflation dataset.
- Removed unnecessary columns.
- Renamed column names.
- Converted wide-format yearly data into long format using Unpivot.
- Corrected data types.
- Integrated country-region mapping data.
- Created calculated fields for adjusted inflation rates and categories.

---

## 📊 Dashboard Features

### Page 1: Executive Dashboard

- KPI cards for average, maximum, and regional inflation analysis.
- Global inflation trend from 1980–2024.
- Country-wise inflation comparison.
- Inflation category distribution.
- Interactive filters for country, region, and year.

### Page 2: Country & Regional Analysis

- Geographic visualization of countries by region.
- Inflation rate distribution analysis.
- Top inflation countries comparison.
- Regional insights using interactive visuals.

---

## 📈 Key Insights

- Inflation trends vary significantly across countries and years.
- Some countries experience extremely high inflation compared to the global average.
- Inflation behavior differs across global regions.
- Interactive filters enable detailed country-level analysis.
